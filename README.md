### 1. Пошаговое описание работы алгоритма

**Алгоритм локального поиска для задачи о независимом множестве:**

1. **Инициализация**: Начинаем со случайного независимого множества (может быть пустым)
2. **Итеративное улучшение**: 
   - Просматриваем все вершины графа
   - Для каждой вершины проверяем, можно ли её добавить в текущее множество
   - Вершину можно добавить, если она не смежна ни с одной вершиной из текущего множества
3. **Добавление вершин**: Добавляем все подходящие вершины в множество
4. **Завершение**: Когда нельзя добавить больше вершин, возвращаем найденное множество

### 2. Оценка временной сложности
**O(V²)**, где V - количество вершин в графе

### 3. Объяснение временной сложности
- Проверка каждой вершины на возможность добавления: O(V)
- Для каждой вершины проверяем смежность со всеми вершинами в текущем множестве: O(V)
- В худшем случае: O(V × V) = O(V²)

### 4. Краткий ответ на саму тему
Локальный поиск для задачи о независимом множестве начинается со случайного решения и итеративно улучшает его, добавляя вершины, не смежные с текущим множеством. Алгоритм находит приближенное решение за полиномиальное время. Качество решения зависит от структуры графа и начального приближения.

### 5. Примеры вводных данных и выводимые данные

**Входные данные**: Граф с 12 вершинами (матрица смежности)

**Выводимые данные**: Найденное независимое множество и его размер
